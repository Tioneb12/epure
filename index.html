Politique de Confidentialité

Dernière mise à jour : février 2026

Avertissement

Épure est un projet personnel, né de la curiosité et de l'envie d'expérimenter. C'est un petit système en JavaScript vanilla, sans prétention. Les suggestions d'amélioration et les signalements de bugs sont les bienvenus, mais c'est un side project — le temps que j'y consacre reste limité. Pas de support dédié, pas de roadmap gravée dans le marbre. Juste un outil fait avec soin, que vous êtes libre d'utiliser tel quel.

Engagement

Ma philosophie

Épure est conçue avec un principe fondamental : vos données restent les vôtres. L'extension fonctionne entièrement en local sur votre navigateur. Aucune donnée n'est collectée, transmise ou stockée sur un serveur distant.

Aucun compte n'est requis. Aucune inscription. Aucun tracking. Aucune analytique.

Données

Ce que je ne collecte pas

  • Aucune donnée personnelle (nom, email, profil LinkedIn)
  • Aucun contenu de votre feed LinkedIn
  • Aucune information de navigation ou d'historique
  • Aucune métrique d'utilisation ou télémétrie
  • Aucun cookie tiers ou pixel de suivi
Stockage

Données stockées localement

Épure utilise exclusivement les API de stockage du navigateur pour sauvegarder vos préférences :

chrome.storage.sync

Votre configuration : filtres activés, poids, seuils, mode d'affichage, mots-clés personnalisés, liste d'auteurs de confiance. Ces données sont synchronisées entre vos appareils via votre compte Chrome, sans transiter par un serveur externe.

chrome.storage.local

Compteurs de session (nombre de posts filtrés et signalés). Ces données restent strictement sur votre appareil.

Aucune de ces données ne quitte votre navigateur vers un service tiers.

Permissions

Permissions utilisées

Accès à linkedin.com

L'extension injecte un script de contenu dans les pages LinkedIn pour analyser le feed et appliquer les traitements visuels (floutage, masquage, mise en avant). L'analyse se fait entièrement côté client, en mémoire. Aucun texte n'est envoyé à un serveur.

storage

Pour sauvegarder votre configuration et vos compteurs localement.

commands

Pour le raccourci clavier Alt+S (Signal Only).

Réseau

Communications réseau

Épure n'effectue aucun appel réseau. Aucune requête HTTP, aucune API externe, aucun service d'analytique. L'extension fonctionne intégralement hors-ligne une fois installée.

Vos droits

Contrôle total

  • Exportez votre configuration à tout moment depuis l'onglet Config
  • Réinitialisez tous vos paramètres en un clic
  • Désinstallez l'extension pour supprimer toutes les données associées

Aucune donnée résiduelle ne persiste après la désinstallation.
